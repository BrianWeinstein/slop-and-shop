# **App Name**: Slop & Shop

## Core Features:

- Vertical Video Playback: Play videos in a full-screen, vertical format optimized for mobile viewing.
- Snap-Scrolling: Implement seamless snap-scrolling functionality for easy video browsing.
- Looping Placeholder Videos: Use 15-second looping videos as placeholders for continuous playback.

## Style Guidelines:

- Dark mode background (#121212) to enhance video focus and reduce eye strain.
- Primary color: Electric purple (#BE72FF) for interactive elements and highlights.
- Accent color: Bright cyan (#72F3FF) for call-to-action elements.
- Font: 'Inter', sans-serif, for clear and modern UI text.
- Full-screen video display with minimal UI elements to maximize viewing space.
- Smooth transitions between videos during snap-scrolling for a fluid user experience.
